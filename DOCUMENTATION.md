# Job Portal Project Documentation

## Project Overview

The Job Portal Website is a web application developed using Django framework. It allows employers to post jobs and job seekers to apply online.

---

# Modules

## Employer Module

Features:
- Employer Registration
- Employer Login
- Add Jobs
- Edit Jobs
- Delete Jobs
- View Applicants
- Update Applicant Status

---

## Job Seeker Module

Features:
- Job Seeker Registration
- Login
- Search Jobs
- Apply for Jobs
- Upload Resume
- View Applied Jobs

---

## Contact Module

Features:
- Contact Form
- Email & Phone Clickable Links
- Google Map Integration
